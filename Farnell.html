Farnell shopping list

This list is to help us save time and postal fees when buying stuff. If you plan to go to the shop or place an order, please set a deadline and add warning so no one else will place the same order.

Use the following table to add stuff you would like to buy from Farnell.

We have achieved platinum membership on shared account, so ask for it before taking an order -> better prices

There are two ways how to order from Farnell.

  1. Direct order from farnell.com - 2-3 days delivery.
  2. Order through rothsware.cz - Sometimes it takes way too long, but usualy 2-5 days.

Everything what you need to do is download your shopping basket in CSV format and send this file (+name, adress) to obchod@rothsware.cz They send you goods with invoice. Then you have approx. 14 days to pay.

Main advantage is that if something went wrong, you don't need to bother with complaining to UK.

Remember that all prices are WITHOUT VAT!!!

Be carefull - for items on US stock you need to pay extra 680CZK per order. Also check if item is hazardous goods or not. If so, sometime it require special handling and is delivered separately. It means some extra cost for shipping.
