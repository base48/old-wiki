Beehive weight measuring provides interesting data for analysis of the health of the colony. This project is an incremental prototyping platform.

Implementation v1

Current implementation works on a CAN bus with "nodes", where each node acquires data from 2 load cells. CAN bus master on Raspberry Pi 3 through SPI controller. Data transmitted to server.

Status: Done

Implementation v2

Low power version with LoRaWAN connectivity.

Status: Bought components

Plan

Completely remake on different platform. This project is subset of future release (more universal).
