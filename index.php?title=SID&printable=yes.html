Goal

  • Provide complete opensource stack for building and running a SID monitor.
  • Provide documentation for people interested in running their own monitor.
  • Build a network of SID monitors.

Hardware

Loop antenna, preamp, sound card, PC.

Software

  • https://github.com/sorki/sidc
  • https://github.com/sorki/sidc-gui

Installation

Both tools are packaged in Fedora and can be installed by running: 

yum install sid sidc-gui

Pre-configured live image of Fedora 20 is available at 

http://48.io/~rmarko/random/SID.iso
