Velobot

Open hardware bike computer based on singleboard PC like RaspberryPi or BeagleBone that allows for tinkering and lets you keep and analyze your bike rides without uploading your data to 3rd party sites or using proprietary software.

Also a platform for

    • data acquisition
    • research
    • robotics

Hardware overview

  • Single-board computer (RaspberryPi, Cubieboard)
  • MCU
  • Display
  • GPS module
  • GSM module
  • IMU (Accelerometer, Magnetometer, Barometer)
  • RTC
  • Power supply

Software overview

  • Linux
  • ROS (Robot Operating System)
  • avahi
  • chrony
  • gpsd
  • hostapd
  • NetworkManager
  • PostgreSQL (PostGIS)

Subprojects

  • Bicycle overtaking experiment

  • GPS Tracker

Prototype #2

Hardware

  • Raspberry Pi 2
  • FRDM-KL25Z (ARM Cortex-M0)
  • GPS
    • uBlox NEO-6M
  • Display
    • 128x32 OLED
  • IMU
    • MPU-6050 (accelerometer + gyro)
    • HMC5883L (magnetometer)
    • MS5611 (barometer)
  • RTC
    • DS1307 RTC
  • Power
    • Lipo Rider Pro
    • 2000 mAh Li-Po battery

Software

  • Fedora / Debian
  • systemd
  • ROS Indigo
    • rosserial_mbed
  • avahi
  • chrony
  • gpsd
  • hostapd
  • PostgreSQL
  • Provisioned with rpi_kickstart

Custom kernel build with RTC_HCTOSYS enabled so the system starts with correct clock.
